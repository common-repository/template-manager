== Description ==

Plugins allow you to create a multiple custom template for WordPress Gutenberg Block. Manage templates from one place and instantly see changes without digging into each individual page/post of the layout. It is automatically applied changes to new or existing posts and pages.

Create a custom set of pre-defined blocks and use them at any place of your site. Forget about back and forth page updating - It is managed by the plugin and yes, you can easily edit them whenever you wish!

It is a collection of multiple defaults Gutenberg Block. Globally manage your website page/ post block.

= How to use Template Manager for Gutenberg Block? =

Let’s say you created a block with contact details eg. Phone, Address, Email, Contact Form and used that block in multiple locations (pages) across your site, then you decided to change Phone or Email address.

All you have to do is edit your original template and all “ Template Manager  ” blocks will be updated at the very same moment! So your new contact details will be updated automatically on your site.

= Key Features =

* Create Multiple Custom Template for Gutenberg Block
* After updating the template structure will also change for the exiting pages and post.
* Manage the Gutenberg block template by using the back-end interface.
* Create Template and use on multiple pages/post and when you will update that template it will be updated across all pages where it was used! You will never go back after using this block.

= INSTALL INSTRUCTIONS =
* Upload the folder "template-manager" folder to the /wp-content/plugins/ directory of your site.
* Activate the plugin through the ‘Plugins’ menu in wordpress.
